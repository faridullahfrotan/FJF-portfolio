# 🌐 Personal Portfolio - Portwind

A modern, responsive personal portfolio website built using HTML, Tailwind CSS, and JavaScript.

---

## 🚀 Live Preview
👉 https://your-live-demo-link.com

---

## 📌 Features

- Clean and modern UI design  
- Fully Responsive (Mobile, Tablet, Desktop)  
- Light Mode & Dark Mode toggle  
- Smooth animations and transitions  
- Navigation bar with sections:
  - Home  
  - About Me  
  - Services  
  - Projects  
  - Contact Me  
- Well-structured and maintainable code  

---

## 🛠️ Technologies Used

- HTML5  
- Tailwind CSS (CDN)  
- JavaScript (Vanilla JS)

## 🎨 Sections Overview

### Home
Hero section with introduction, profile image, and short description.

### About Me
Personal information, skills, and experience.

### Services
Web design, frontend development, and UI/UX improvements.

### Projects
Showcase of portfolio projects with interactive cards.

### Contact Me
Contact form or social media links.
